    여는 태그와 닫는 태그 둘중 하나만 바꿔도 나머지 하나도 자동으로 바뀜.

    중요도
    75%

    코드를 작성하다 보면 괄호가 많아서 이게 어디부터 시작되는 괄호인지 헷갈릴때가 있는데
    괄호마다 색을 다르게 지정해줘서 보기 편하게 해줌.

    중요도
    50%
  3. Community Material Theme

    VSC의 테마를 바꿔줌. 가장 인기있는 테마.

    중요도
    10%
  4. CSS Peek

    HTML에서 CSS파일을 쉽게 수정할 수 있게함. 버그가 많음.

    중요도
    45%
  5. HTML CSS Support

    HTML id and class attribute completion for Visual Studio Code.

    중요도
    70%
  6. HTML to CSS autocompletion

    HTML파일에서 지정한 클래스나 아이디 명을 CSS파일에서 자동완성 할수있음.

    중요도
    80%
  7. indent-rainbow

    들여쓰기(Tab) 1개(기본 4칸)마다 색을 다르게 하여 줄 구분이 쉬워짐.

    중요도
    80%
  8. Korean(사용법) Language Pack for Visual Studio Code

    VSC의 언어를 한국어로 바꿔줌.

    중요도
    90%
  9. Live Server

    HTML을 실시간으로 브라우저에 띄워줌.

    중요도
    100%
  10. Material Icon Theme

    파일의 아이콘을 바꿔줌.

    중요도
    10%

HTML이란?


HTML이란 Hypertext Markup Language의 약자로 웹페이지를 만들 때 쓴다.

여기서 Markup Language는 자료의 구조를 표현하기위한 언어다.

우리가 웹페이지 안에 글넣고 그림넣고 버튼넣는데 그런 자료가

어디에 위치하는지 그런 것들을 집어넣을수있는 언어가 HTML이다.

HTML 파일 기본 템플릿

HTML 문서는 .html로 끝나도록 저장하시고
모든 HTML 문서는 위의 코드를 가지고 있어야됩니다.
<head> 내부엔 사이트 생성에 필요한 인코딩형식, 사이트제목, 필요한 CSS나 JS파일 등이 들어갈 수 있으며
<body> 내부엔 실제 웹사이트에서 보여줄 글, 그림 등을 적어주시면 됩니다.

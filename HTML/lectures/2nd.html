HTML은 <태그>로 작성합니다.

기본 태그 목록

<p>글 본문</p>

<h1>글 제목</h1>

<img src="이미지 경로">

<a href="">링크</a>

<button>버튼</button>

<ul><li>리스트</li></ul>

<ol><li>리스트</li></ol>

  • 태그를 열었으면 </태그명>으로 닫아야합니다. <img>이런 태그와 같이 닫지 않아도 되는 태그도 몇개 있습니다.
  • 특정 태그는 안에 href=" ", src=" "와 같은 속성을 집어넣어서 추가정보를 기입가능합니다.


  • 용도에 맞는 태그를 사용해야합니다.

    용도에 맞는 태그를 쓰는게 강제되는 사항은 아닙니다.

    하지만 HTML은 마크업언어기 때문에 자료의 구조를 쉽게 파악할 수 있어야하는데

    그럴 때 용도에 맞는 태그 사용하는게 도움이 되고 용도에 맞는 태그를 사용하면

    더욱 '웹표준'에 맞는 웹을 만들어낼 수 있습니다.



더 많은 태그

HTML에는 정말 많은 태그가 있고 용도도 다양합니다.

그렇다면 이 많은 태그를 다 외워야할까요?

HTML은 계속해서 업데이트 되면서 다양한 태그가 생기기도 하고 사라지기도 합니다.

따라서 우리는 가장 많이쓰는 몇가지 태그만 외우고

나머지는 필요할때 구글에서 검색해서 쓰면 됩니다.

가장 많이 쓰는 HTML태그 : https://www.advancedwebranking.com/



간단한 스타일 넣는 법



<p style="font-size : 20px; color : red;"> 글자 </p>

거의 모든 태그는 style=""이라는 속성을 열 수 있습니다.

거기 안에 스타일이름 : 스타일값;형식으로 스타일을 넣으면 됩니다.



자주 사용하는 글자 스타일들

font-size : 20px; //글자의 크기를 조절함
font-family : 'gulim'; //글씨체를 바꿈
color : black; //글자의 색을 바꿈
letter-spacing : -1px; //자간을 조절함
text-align : center; //글자를 정렬함
font-weight : 600; //글자의 굵기를 조절함


텍스트의 일부만 스타일을 변경하고 싶을 때



<p>1학년 9반 조윤성은 <span style="color : red;">멍청한</span>인간입니다.</p>

<span> 이라는 태그로 감싼 뒤에 스타일을 주면 됩니다.

<span> 태그는 가끔 글자 일부를 싸매고 싶을 때 사용하는 큰 의미없는 태그입니다.



(참고) span 태그는 display : inline 이라는 스타일 속성을 내포하고 있으며,

display : inline을 가지고 있는 요소는 폭, 높이 등을 단독으로 결정할 수 없습니다

폭, 높이를 주고싶으면 얘를 감싸고 있는 <p>에 주십시오

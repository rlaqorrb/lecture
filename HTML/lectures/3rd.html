<p style="font-size : 20px; color : red;"> 글자 </p>

지금까지 우리는 글자에 이런식으로 스타일을 넣었습니다.

그런데 style 속성 안에 들어갈 코드가 길어지면

HTML이 더러워지기 때문에 이걸 CSS 파일로 빼서 작성할 수 있습니다.

CSS파일을 만든 후 HTML에 첨부하기

우선 작업폴더에 .css로 끝나는 파일 아무거나 하나 만들고

HTML 파일로 돌아와서

<link href="css파일경로" rel="stylesheet">

<head>태그 안에 <link>태그를 넣고

href라는 속성에 css파일 경로를 지정해놓으면 css파일 이용가능합니다.



(참고)

  • 만약 style.css 라는 파일을 html파일과 같은 폴더에 만들면

    "css파일경로"에 "style.css"라고 쓰면 됩니다.

  • 만약 css라는 폴더안에 style.css파일이 있으면

    "css/style.css"라고 쓰시면 됩니다.

  • html파일이 있는 폴더의 이전 폴더에 style.css파일이 있으면

    "../style.css"라고 쓰시면 됩니다.

이런걸 '상대경로'라고 합니다.




여기까지 하면 css파일에서 스타일 작성이 가능합니다.

정확한 방법은

1. HTML에서 클래스를 만들어줍니다.

<p class="class이름">글자</p>

class 속성을 열고 클래스명을 집어넣어줍니다.

이때 클래스명은 아무거나 지어도 되는데

이 클래스가 어떤 것을 수식하는지 쉽게 알수있게 이름을 지어줍니다.

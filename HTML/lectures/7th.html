이전 강의의 숙제 강좌입니다.

몇가지 팁을 주고 풀이까지 합니다.






팁0) 레이아웃은 항상 박스부터 만들고 시작하면 됩니다.

만들고 싶은 레이아웃 위에 박스를 그립니다.

이런식으로 그리고 이 박들을 저부 <div>로 구현하면 됩니다.

그리고 안에 글넣고 그림넣고 하면 끝입니다.

여기서 기억해야될게 <div>는 그냥 쓰면 display : block 때문에 위아래로 배치됩니다.

좌우로 나란히 배치하고 싶으면 float, 혹은 inline-block 쓰면 됩니다.






팁1) margin, padding 속성은 상하좌우 마진을 한꺼번에 줄 수 있습니다.

.box {
margin : 10px auto 30px auto;
}

margin 이나 padding 속성에 4개의 값을 동시에 집어넣을 수 있는데 4개 값은 각각 상, 우, 하, 좌측을 의미합니다.






팁2) PC레이아웃을 만드실 때 항상 container 또는 wrap박스를 만들어놓는게 좋습니다.

<div class="container">
  <div class="left"></div>
  <div class="right"></div>
</div>

container박스엔 항상 width를 지정해놓는게 좋습니다.

그래야 나중에 브라우저화면이 축소되어도 내부 div 박스들이 찌그러지지 않습니다







숙제 해설

먼저 전체 레이아웃을 감싸주는 <div>부터 만들어줍니다.

<div class = "container">

</div>

width는 800px 정도 주면 될거같습니다.



그다음으로는 만들고자 하는 레이아웃을 박스로 그려봅니다.

박스로 그리고 div를 이용해 만들어줍니다.

<div class = "container">
  <div class = "blog-content">

  </div>
  <div class = "blog-img">
    <img>
  </div>
</div>
.blog-content {
  width : 80%;
  height : 200px;
  background : #eee;
}

.blog-img {
  width : 20%;
  height : 200px;
  background : yellow;
}

만들 레이아웃을 크게 보면 왼쪽에 내용과 오른쪽에 사진이 있습니다.

그래서 큰 2개의 박스로 먼저 나누어 주고

각각 width를 80%, 20% 줬습니다.

배경과 높이는 처음에 만들기 쉬우라고 준겁니다.

여기까지 하면 아래와 같이 만들어질겁니다


예시

div박스는 display : block을 가지고 있어서 위아래로 배치됩니다.



따라서 요소를 가로로 배치하는 float나 inline-block을 쓰면 됩니다.

여기서는 좀 더 쉬운 float를 사용하겠습니다.

.blog-content {
  width : 80%;
  height : 200px;
  background : #eee;
  float : left;
}

.blog-img {
  width : 20%;
  height : 200px;
  background : yellow;
  float : left;
}

이렇게 float : left를 각각 주면

아래와 같이 요소가 좌우로 나란히 배치됩니다.


예시


그리고 이제 이미지를 넣어줍니다.

<div class = "container">
  <div class = "blog-content">

  </div>
  <div class = "blog-img">
    <img src = "apple.jpg" width = "100%">
  </div>
</div>

이렇게 이미지를 넣어줍니다.

이미지의 속성에는 width가 있는데 css와 비슷하게 사용 가능합니다.



그러면 이제 왼쪽 박스를 디자인해 봅시다.

<div class = "container">
  <div class = "blog-content">
  <div>
    <img src = "lion.jpg" class = "blog-profile">
    <h4>이름</h4>
    <p>날짜</p>
  </div>
  <div> </div>
  <div class = "blog-img">
    <img src = "apple.jpg" width = "100%">
  </div>
</div>

왼쪽 박스는 크게 (이름과 사진이있는)윗쪽 박스와

(내용있는)아랫쪽 박스로 나누어집니다.

따라서 div박스를 2개 만들어줍니다.

윗쪽 박스에는 이미지와 이름, 날짜가 들어가므로 그것도 만들어 줍니다.

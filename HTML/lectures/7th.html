이전 강의의 숙제 강좌입니다.

몇가지 팁을 주고 풀이까지 합니다.






팁0) 레이아웃은 항상 박스부터 만들고 시작하면 됩니다.

만들고 싶은 레이아웃 위에 박스를 그립니다.

이런식으로 그리고 이 박들을 저부 <div>로 구현하면 됩니다.

그리고 안에 글넣고 그림넣고 하면 끝입니다.

여기서 기억해야될게 <div>는 그냥 쓰면 display : block 때문에 위아래로 배치됩니다.

좌우로 나란히 배치하고 싶으면 float, 혹은 inline-block 쓰면 됩니다.






팁1) margin, padding 속성은 상하좌우 마진을 한꺼번에 줄 수 있습니다.

.box {
margin : 10px auto 30px auto;
}

margin 이나 padding 속성에 4개의 값을 동시에 집어넣을 수 있는데 4개 값은 각각 상, 우, 하, 좌측을 의미합니다.






팁2) PC레이아웃을 만드실 때 항상 container 또는 wrap박스를 만들어놓는게 좋습니다.

<div class="container">
  <div class="left"></div>
  <div class="right"></div>
</div>

container박스엔 항상 width를 지정해놓는게 좋습니다.

그래야 나중에 브라우저화면이 축소되어도 내부 div 박스들이 찌그러지지 않습니다




꼭 먼저 고민 해보고 풀이를 보셔야됩니다.




숙제 해설

먼저 전체 레이아웃을 감싸주는 <div>부터 만들어줍니다.

<div class = "container">

</div>

width는 800px 정도 주면 될거같습니다.



그다음으로는 만들고자 하는 레이아웃을 박스로 그려봅니다.

박스로 그리고 div를 이용해 만들어줍니다.

<div class = "container">
  <div class = "blog-content">

  </div>
  <div class = "blog-img">
    <img>
  </div>
</div>
.blog-content {
  width : 80%;
  height : 200px;
  background : #eee;
}

.blog-img {
  width : 20%;
  height : 200px;
  background : yellow;
}

만들 레이아웃을 크게 보면 왼쪽에 내용과 오른쪽에 사진이 있습니다.

그래서 큰 2개의 박스로 먼저 나누어 주고

각각 width를 80%, 20% 줬습니다.

배경과 높이는 처음에 만들기 쉬우라고 준겁니다.

여기까지 하면 아래와 같이 만들어질겁니다

div박스는 display : block을 가지고 있어서 위아래로 배치됩니다.



따라서 요소를 가로로 배치하는 float나 inline-block을 쓰면 됩니다.

여기서는 좀 더 쉬운 float를 사용하겠습니다.

.blog-content {
  width : 80%;
  height : 200px;
  background : #eee;
  float : left;
}

.blog-img {
  width : 20%;
  height : 200px;
  background : yellow;
  float : left;
}

그 이유는 h4와 p도 display : block을 가지고 있기때문입니다.

예시

여기서 또 문제가 보이는데 블로그 글과 내용이 위로 가있습니다.

그 이유는 프로필 사진과 이름이 float때문에 떠있는 상태라서

그 밑에 있는 블로그 글과 내용이 위에 아무것도 없다고 생각해서 위에 붙어있는겁니다.


저번 강의때 float를 쓰고 난 다음 쓰는것이 있다고 했습니다.

<div style = clear : both;>
  <h4 style = "margin : 8px">이름</h4>
  <p style = "margin : 8px">이름</p>
</div>

이렇게 clear : both를 주면 제자리를 찾을겁니다.

그러고 나서 뒤에 배경을 지우고

전체 박스를 가운데 정렬하면 쓸만한 블로그 레이아웃이 만들어 지겠죠.


전체 박스 가운데 정렬은 margin-left : auto; margin-right : auto;를 쓰면 됩니다.




따라서 최종 코드는 아래와 같고

<div class = "container">
  <div class = "blog-content">
  <div>
    <img src = "lion.jpg" class = "blog-profile">
    <div class = "blog-name">
      <h4>이름</h4>
      <p>날짜</p>
    </div>
  </div>
  <div style = clear : both;>
    <h4 style = "margin : 8px">이름</h4>
    <p style = "margin : 8px">이름</p>
  </div>
  <div class = "blog-img">
    <img src = "apple.jpg" width = "100%">
  </div>
</div>
.container{
width : 800px;
margin-left : auto;
margin-right : auto;
}

.blog-content{
width : 80%;
height : 200px;
float : left;
}

.blog-img{
width : 20%;
height : 200px;
float : left;
}

.blog-profile{
width : 50px;
display : block;
float : left;
}

.blog-name{
float : left
}

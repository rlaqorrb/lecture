웹페이지에서 레이아웃을 만들때는 네모 박스를 이용합니다.

네모 박스는 <div>태그를 이용합니다.

<div></div>

이렇게 쓰면 1차원의 박스가 생깁니다. (높이가 없는 선)

이 박스를 보이게 하려면 div태그 안에 내용을 쓰거나 CSS로 높이를 줘야합니다.





박스 디자인에 많이 사용하는 CSS속성

.box {
  margin : 20px;
  padding : 30px;
  border : 1px solid black;
  border-radius : 5px;
}

margin은 바깥 여백,

padding은 안쪽 여백,

border는 테두리 (차례로 두께, 선의 종류, 색상),

border-radius는 테두리를 둥글게 처리합니다.

가운데 정렬은 저번에 했던 이미지처럼
display : block; margin-left : auto; margin-right : auto 이렇게 주면 됩니다.

저번에도 말했듯이 모든 것을 다 외우려고 하지말고 필요할때 구글에서 찾아씁시다.






margin과 padding, border를 원하는 방향에만 줄 수 있습니다.

.box {
  margin-top : 20px;
  padding-left : 30px;
  border-bottom : 1px solid black;
}

top, right, bottom, left 중 원하는 곳에만 여백을 줄 수 있습니다.

(참고) margin은 음수도 가능합니다. -20px 이런 식입니다.

(참고) margin : 5px 6px 7px 8px; 이렇게 띄어쓰기를 이용해 작성하면

차례로 상 우 하 좌 margin을 5,6,7,8px 한번에 줄 수 있습니다.






display : block이 내장되어있는 div박스

.box {
  display : block;
}

모든 div, h1, li 등은 display : block 속성을 기본적으로 가지고있습니다.

그래서 이런 태그를 그냥 사용하면 한 행을 전부 차지하게 됩니다.

물론 display 속성을 다른 것으로 바꿀 수 있습니다.

display : inline, inline-block, flex 등 여러가지가 있습니다.






inherit되는 스타일


font-size, color, font-family, text-align 이런 속성들은

부모 태그에 부여하면 거기 안에 있는 태그들까지 전부 상속됩니다.

영어로 inherit 된다고 합니다.

안에 글자들이나 태그들이 많을 경우 전부 font-size를 작성안해도

부모태그에 한번에 작성하고 끝낼 수 있으니 편리합니다.

다 inherit되는건 아니고 글자와 관련된 스타일들이 주로 inherit 됩니다.
